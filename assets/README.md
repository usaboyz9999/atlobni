## ملاحظة

لتشغيل التطبيق محلياً بدون أصول حقيقية، يمكن استخدام الأوامر التالية لإنشاء أصول placeholder:

```bash
npx expo install expo-asset
```

أو تشغيل:
```bash
npx create-expo-app --template blank
```
ونسخ مجلد assets منه.
